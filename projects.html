Orbital Mechanics Simulation — L2 Lagrange Point

Simulated the orbital motion of a satellite near the Sun–Earth L2 Lagrange point. Estimated the L2 location from orbital mechanics relations, then propagated the satellite's trajectory using numerical integration in the rotating frame.

  • Estimated L2 position from orbital mechanics relations
  • Initial implementation with Euler integration; refined to more stable methods
  • Simulated trajectories in the rotating reference frame
  • Modelled the restricted 3-body problem (Earth–Sun)
  • Visualized motion to confirm L2 behavior

MATLAB — Euler Integration (gravity forces)

% Force components (gravity)
Fx(i) = -G*Me*m * x(i) / (x(i)^2 + y(i)^2)^(1.5);
Fy(i) = -G*Me*m * y(i) / (x(i)^2 + y(i)^2)^(1.5);

% Euler integration
vx(i+1) = vx(i) + Fx(i)/m * dt;
vy(i+1) = vy(i) + Fy(i)/m * dt;

x(i+1) = x(i) + vx(i) * dt;
y(i+1) = y(i) + vy(i) * dt;

MATLAB — Predictor-corrector integration

% Predictor step
x_pred = x(i) + deltat*uNow;
y_pred = y(i) + deltat*vNow;
r_pred = sqrt(x(i)^2+y(i)^2);

u_pred = uNow + deltat*(-r_pred^-3)*x(i);
v_pred = vNow + deltat*(-r_pred^-3)*y(i);

% Corrector step
xCorr = (x(i) + x_pred)/2 + deltat/2 * u_pred;
yCorr = (y(i) + y_pred)/2 + deltat/2 * v_pred;

rCorr = sqrt(x_pred^2+y_pred^2);
uCorr = (uNow+u_pred)/2 + deltat/2 * (-rCorr^-3) * x_pred;
vCorr = (vNow+v_pred)/2 + deltat/2 * (-rCorr^-3) * y_pred;

Aircraft Range & Service Ceiling Analysis

Computed aircraft range, drag polar, and service ceiling for a transonic aircraft using standard performance equations. The drag model included parasite drag, induced drag, and wave drag (with drag divergence), enabling realistic cruise and ceiling estimation.

  • Modelled parasite, induced, and wave drag contributions
  • Computed drag divergence Mach number from sweep and thickness
  • Calculated cruise range from drag polar
  • Determined service ceiling from excess power curves

MATLAB — Drag polar computation

% Induced drag
e      = 0.95;
K      = 1./(pi.*e.*wing(6));
indDrag = K.*CL.^2;

% Wave drag (drag divergence model)
machDD = (0.95 - CL./(10.*cos(wing(9))^2) ...
          - wing(10)./cos(wing(9))) ./ cos(wing(9));
machCr  = machDD - (0.1./80)^(1/3);
wavDrag = 20.*max(0,(mach-machCr)).^4;

% Total drag coefficient
totalCD = parDrag + indDrag + wavDrag;
